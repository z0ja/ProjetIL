# ProjetIL
Projet d'ingénieurie logicielle

Pour utiliser le projet en entier il vous faut être connecté au réseau de l'université (VPN) et lancer les commandes suivantes en arrière plan depuis la racine du projet:
  `node server.js`
  `node room/index.js`
  `node tchat/index.js`

la recherche de vidéos youtube ne fonctionnera pas sans le fichier .env dans la racine 
la création de rooms ne fonctionne que si vous êtes connectés avec un compte
